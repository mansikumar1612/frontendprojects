Frontend Development

Frontend development is the development of the graphical user interface of a website, through the use of HTML,CSS and Javascript. so that users can view and interact with that website.

  • Html: The hypertext markup language or HTML is the standard markup language for documents designed to be displayed in a web browser.


  • Css: cascading style sheets (CSS) is a style sheets language used for describing the way how can we style and design our website by giving deifferent properties




    1. Pure Css

    2. Bootstrap (Framework)

    3. Tailwind (Framework)

    4. Sass/preprocessor


  • Javascript: Javascript often abbreviated JS, is a programming language that is one of the core technologies of the world wide web, alongside HTML and CSS. It is used to add functionality in the website.




    1. Vanilla JS

    2. jQuery (library)

    3. React (framework)

    4. Angular (framework)

    5. Vue (framework)

    6. Typescript (Preprocessor)
